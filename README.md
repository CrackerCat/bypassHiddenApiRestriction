## BypassHiddenApiRestriction
This is a library that can bypass the restrictions on non-SDK interfaces on Android 9-12.

## Usage
```
import com.wind.hiddenapi.bypass.HiddenApiBypass

HiddenApiBypass.startBypass()
```
